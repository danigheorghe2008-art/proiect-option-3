# proiect-option-3
proiect for 11 november

Option three
Implement the "Cost Accounting" application. The main task of the application is to store information about the user's expenses.
The application interface should provide the following features:
Adding expense categories.
Deleting expense categories.
Adding a specific expense. The following information should be provided:
Name of the expense;
Date;
Category.
Saving expenses to a file.
Loading expenses from a file.
Generating reports by the following parameters:
By date;
By name;
By category;
Display the maximum expense in each category;
Display the maximum expense in the specified time period;
Displays the minimum expense in each category;
Display the minimum expense in the specified time period.
Correct use of design patterns, SOLID principles, and testing mechanisms during the implementation of the task will allow you to get a higher score.
